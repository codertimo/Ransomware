##Ransomware를 활용한 Locker

이를 복호화 할수있는 키를 판매하는 바이러스가 있습니다.

이를 Ransomware 바이러스라고 하고, 저희는 Ransomware를 활용하여

자신이 숨기거나 보호하고 싶은 파일들을 암호화 하여

다른사람들의 접근을 예방하고, 비정상적인 접근에 대비할 수 있습니다.
